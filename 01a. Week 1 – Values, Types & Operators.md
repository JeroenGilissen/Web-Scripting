# Week 1 – Values, Types & Operators

## Wat is JavaScript?

- **Interpreted:** code wordt direct uitgevoerd (niet gecompileerd zoals C)
- **Dynamisch getypeerd:** type-check gebeurt tijdens uitvoering (niet bij compilatie zoals Java)
- **Weakly typed:** datatypes worden automatisch omgezet (**implicit conversion**)
- Gebaseerd op de **ECMAScript**-standaard (vanaf 2016 aangeduid met het jaartal)

---

## Variabelen & Constanten

### `let` — variabele

```js
// Declaratie & initialisatie in 1 keer
let aantal = 1;

// Declaratie & initialisatie verspreid
let waarde;
waarde = 1;

// Meerdere declaraties op 1 regel
let a, b = 1, c;
```

### `var` — variabele

> ⚠️ **Niet gebruiken.** Gebruik altijd `let` of `const`.

### `const` — constante

Waarde kan **niet** gewijzigd worden na toekenning.

```js
// Declaratie & initialisatie in 1 keer
const een = 1;

// Meerdere constanten
const twee = 2, drie = 3;
```

## Type: number

Geen onderscheid tussen komma- en gehele getallen.

```js
let number = 13;
let decimalNumber = 3.14;
let avogadroNumber = 6.0221e23;

let teGrootGetal = 1e1000; // → Infinity

if (teGrootGetal === Infinity) {
  console.log("Infinity!");
}

console.log(1 / teGrootGetal); // → 0
console.log(0 / 0);            // → NaN
```

**Bewerkingen:** `+`, `-`, `*`, `/`, `%`

```js
let number = 13;
console.log(number / 2); // → 6.5
console.log(number % 2); // → 1  (rest na deling)
```

**Cast naar number:**

```js
let input = prompt("Geef een getal");
let number = Number(input);
```

## Type: string

**Single & double quotes** (zelfde effect):

```js
let welcome = "Hello\nWorld"; // \n = nieuwe lijn
```

**Backticks** (template literals) — variabelen inlassen met `${}`:

```js
let getal = 12;
console.log(`${getal} + 1 = ${getal + 1} !`); // → 12 + 1 = 13 !
console.log(`half of 100 is ${100 / 2}`);       // → half of 100 is 50
```

**Geen char-type** — een enkel teken is ook een string:

```js
let welcome = "Hello\nWorld";
let eersteSymbool = welcome[0];
console.log(eersteSymbool);          // → H
console.log(typeof eersteSymbool);   // → string
```

**Concatenatie met `+`:**

```js
console.log("a" + "bc"); // → abc
```

**Cast naar string:**

```js
let getal = 12;
let result = String(getal); // → "12"
```

## Type: boolean, undefined, null

### `boolean`

```js
let goedGekeurd = true;

// Negatie met !
if (!goedGekeurd) {
  console.log("Niet goedgekeurd");
}

// AND en OR
if (name == "tim" && age > 2) {
  console.log("Hallo Tim!");
}
```

### `undefined`

Variabele gedeclareerd maar nog geen waarde toegekend:

```js
let getal;
console.log(typeof getal); // → undefined
```

### `null`

Bewust lege waarde:

```js
let a = null;
```

## Wrappers

Primitives (`string`, `number`) hebben automatisch toegang tot methodes via wrappers (`String`, `Number`):

```js
let name = "Sofie";
console.log(typeof name);              // → string
console.log(name.toUpperCase());       // → SOFIE
// name wordt tijdelijk omgezet naar String-object

let street = new String("Wetstraat");
console.log(typeof street);            // → object
```

## Logica & Vergelijkingen

```js
console.log(1 < 2);              // → true
console.log(1 == 1);             // → true
console.log("aardvark" < "zebra"); // → true  (alfabetisch)
console.log("a" == "jan");       // → false
```

**Ternary operator (inline if):**

```js
let age = 19;
let usertype = age < 18 ? 'minor' : 'adult';
// → 'adult'
```

## Type Coercion (automatische conversie)

JavaScript converteert types automatisch bij bewerkingen:

```js
console.log(8 * null);    // → 0       (number)
console.log("5" - 1);     // → 4       (number)
console.log("5" + 1);     // → "51"    (string! + = concatenatie)
console.log("five" * 2);  // → NaN     (number)
console.log(false == 0);  // → true    (== vergelijkt na conversie)
console.log(false === 0); // → false   (=== vergelijkt ook type!)
```

> 💡 Gebruik **`===`** (strict equality) om onverwachte conversies te vermijden.

### Truthy / Falsy

Niet-boolean waarden worden in een `if` automatisch omgezet:

```js
if ("a") {
  console.log("de string 'a' is truthy"); // wordt uitgevoerd
}

if (!null) {
  console.log("null is falsy");           // wordt uitgevoerd
}
```

**Falsy waarden:** `false`, `0`, `""`, `null`, `undefined`, `NaN`  
**Alles andere is truthy.**

### `||` als standaardwaarde

```js
// Als a truthy → result = a, anders result = b
let result = a || b;

function assignName(name) {
  return name || "unknown";
}

console.log(assignName(null));    // → unknown
console.log(assignName("sofie")); // → sofie
```
