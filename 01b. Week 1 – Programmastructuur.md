# Week 1b – Programmastructuur

## Bindings

Een **binding** verbindt een naam met een waarde. Een variabele is het eenvoudigste voorbeeld.

```js
let i = 1;      // variabele (wijzigbaar)
const j = 12;   // constante (niet wijzigbaar)
```

`const` kan niet gewijzigd worden via assignment operators (`=`, `+=`, `++`, `--`, ...).  
Later leer je ook **function bindings**: de naam van een functie wordt verbonden met uitvoerbare code.

---

### Primitieve data: stack vs heap

Kleine integers worden direct op de **stack** opgeslagen. Grotere getallen en strings worden op de **heap** bewaard, de variabele op de stack houdt dan een **referentie** bij.

```js
let i = 12;       // klein getal → rechtstreeks op stack
let j = 1e33;     // groot getal → heap
let d = 1.22;     // decimaal → heap
let s = "abc";    // string → heap
```

### Reference data

Arrays en objecten staan altijd op de heap:

```js
let a = [1, 2, 3]; // a op stack, [1,2,3] op heap
```

### Garbage collection

Wanneer er geen enkele referentie meer naar een geheugenlocatie bestaat, wordt die automatisch vrijgegeven.

```js
let s = "abc";
s = s + "d";
// "abc" heeft geen referentie meer → wordt opgeruimd
// s verwijst nu naar "abcd"
```

## Sequentie

Code wordt regel per regel van boven naar onder uitgevoerd.

```js
// prompt() geeft altijd een string terug
// Number() zet die string om naar een getal
let theNumber = Number(prompt("Pick a number"));
console.log("Your number is the square root of " + theNumber * theNumber);
```

## If / Else

```js
let theNumber = Number(prompt("Pick a number"));

if (!Number.isNaN(theNumber)) {
  console.log("Your number is the square root of " + theNumber * theNumber);
} else {
  console.log("Hey. Why didn't you give me a number?");
}
```

**If / else if / else:**

```js
let num = Number(prompt("Pick a number"));

if (num < 10) {
  console.log("Small");
} else if (num < 100) {
  console.log("Medium");
} else {
  console.log("Large");
}
```

## Switch

Gebruik `switch` als je één variabele vergelijkt met meerdere vaste waarden.  
Let op: vergeet de `break` niet, anders valt de code door naar het volgende geval (**fall-through**).

```js
let answer = prompt("What is the weather like?");

switch (answer) {
  case "rainy":
    console.log("Remember to bring an umbrella.");
    break;
  case "sunny":
    console.log("Dress lightly.");
    // geen break → valt door naar "cloudy"!
  case "cloudy":
    console.log("Go outside.");
    break;
  default:
    console.log("Unknown weather type!");
    break;
}
```

> ⚠️ Bij `"sunny"` worden zowel `"Dress lightly."` als `"Go outside."` geprint omdat er geen `break` staat.

## While

Herhaalt zolang de voorwaarde `true` is. Voorwaarde wordt **voor** elke iteratie gecontroleerd.

```js
// Print even getallen van 0 t.e.m. 12
let number = 0;
while (number <= 12) {
  console.log(number);
  number = number + 2;
}

// Bereken 2^10
let result = 1;
let counter = 0;
while (counter < 10) {
  result = result * 2;
  counter++;
}
console.log(result); // → 1024
```

---

## Do While

De body wordt **minstens 1 keer** uitgevoerd, voorwaarde wordt **achteraf** gecontroleerd.

```js
let yourName;
do {
  yourName = prompt("Who are you?");
} while (!yourName); // blijft vragen zolang de naam leeg/falsy is

console.log(yourName);
```

---

## For

Compacte schrijfwijze voor loops met een teller: `(initialisatie; voorwaarde; update)`.

```js
// Print even getallen van 0 t.e.m. 12
for (let number = 0; number <= 12; number += 2) {
  console.log(number);
}

// Zoek eerste getal >= 20 deelbaar door 7 (met break)
for (let current = 20; ; current++) {
  if (current % 7 == 0) {
    console.log(current); // → 21
    break;
  }
}
```

## Input in Node.js

In de browser bestaat `prompt()` standaard. In Node.js moet je de library **prompt-sync** installeren.

**Installatie via npm (1 keer per project):**

```bash
npm i prompt-sync
```

Dit maakt een `node_modules`-map en een `package.json` aan:

```json
{
  "dependencies": {
    "prompt-sync": "^4.2.0"
  }
}
```

> 💡 Bij samenwerking: `node_modules` in `.gitignore` zetten. Anderen voeren `npm i` uit om de dependencies te installeren.

**Gebruik in code:**

```js
const prompt = require('prompt-sync')();

function isEenGetal(number) {
  return !isNaN(parseInt(number));
}

let aantal = prompt('Geef het aantal sterren (0-9)');

while (!isEenGetal(aantal) || aantal < 0 || aantal >= 10) {
  console.log("Foute ingave");
  aantal = prompt('Geef het aantal sterren (0-9)');
}

console.log("*".repeat(aantal)); // bv. "****" bij input 4
```
