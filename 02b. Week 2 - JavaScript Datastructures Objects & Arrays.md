# JavaScript Datastructures: Objects & Arrays

## Arrays

### Array Basics

Arrays zijn geordende lijsten met indexed properties (numerieke indices) en named properties.

```javascript
// Array aanmaken
let listOfNumbers = [2, 3, 5, 7, 11];
let emptyArray = [];

// Indexed properties - array elementen benaderen
console.log(listOfNumbers[0]); // 2
listOfNumbers[3] = 3; // element wijzigen

// Named properties - array eigenschappen
console.log(listOfNumbers.length); // 5
console.log(listOfNumbers["length"]); // 5 (alternatieve notatie)
```

### Array Loops

Er zijn verschillende manieren om door arrays te itereren.

```javascript
let numbers = [1, 2, 3, 4];

// 1. Traditionele for loop
for (let i = 0; i < numbers.length; i++) {
    console.log(numbers[i]);
}

// 2. for...of loop
for (let number of numbers) {
    console.log(number);
}

// 3. forEach methode
numbers.forEach((number) => {
    console.log(number);
});
```

### Array Methods

#### Zoeken: indexOf & lastIndexOf

```javascript
let arr = [1, 2, 3, 2, 1];

console.log(arr.indexOf(2));     // 1 (eerste voorkomen)
console.log(arr.indexOf(5));     // -1 (niet gevonden)
console.log(arr.lastIndexOf(2)); // 3 (laatste voorkomen)
```

#### Toevoegen/Verwijderen: push, pop, shift, unshift

```javascript
let numbers = [1, 2, 3];

// pop: verwijder laatste element
let last = numbers.pop(); // numbers = [1,2], last = 3

// push: voeg element toe aan einde
numbers.push(4); // numbers = [1,2,4]
// Alternatief: numbers[numbers.length] = 5;

// shift: verwijder eerste element  
let first = numbers.shift(); // numbers = [2,4], first = 1

// unshift: voeg element toe aan begin
numbers.unshift(0); // numbers = [0,2,4]
```

#### Kopiëren: slice & concat

```javascript
let names = ["jan", "tim", "sofie", "geert", "nele"];

// slice: kopieer deel van array (van index1 tot index3, exclusief index3)
console.log(names.slice(1, 3)); // ["tim", "sofie"]
console.log(names.slice(1));    // ["tim", "sofie", "geert", "nele"]

// concat: voeg arrays/elementen samen
let names2 = ["jan", "tim"];
names2 = names2.concat("sofie");           // ["jan", "tim", "sofie"]
names2 = names2.concat(["geert", "nele"]); // ["jan", "tim", "sofie", "geert", "nele"]
```

### Array Destructuring

Waarden uit een array uitpakken in variabelen.

```javascript
// Basis destructuring
let waarden = [1, 2];
let [a, b] = waarden; // a=1, b=2

// Waarden omwisselen (swap)
[a, b] = [b, a]; // a=2, b=1

// Elementen overslaan
let [first, , third] = [1, 2, 3]; // first=1, third=3

// Met default waarden
let [p, q = 99] = [5]; // p=5, q=99
```

### Spread & Rest Operator (...)

#### Rest Operator - bij destructuring

```javascript
const fruits = ['apple', 'banana', 'orange', 'kiwi'];
const [first, second, ...others] = fruits;

console.log(first);  // "apple"
console.log(second); // "banana" 
console.log(others); // ["orange", "kiwi"]
```

#### Spread Operator - bij array creatie

```javascript
const fruits = ['apple', 'banana', 'orange', 'kiwi'];
const vegetables = ['sprouts', 'broccoli'];
const groceries = [...fruits, ...vegetables];

console.log(groceries);
// ['apple', 'banana', 'orange', 'kiwi', 'sprouts', 'broccoli']
```

#### Rest & Spread met functies

```javascript
// Rest - onbeperkt aantal parameters
function sum(...numbers) {
    let total = 0;
    for (let number of numbers) {
        total += number;
    }
    return total;
}

console.log(sum(1, 2, 3)); // 6

// Spread - array uitpakken naar argumenten
let a = [1, 2, 3, 4];
console.log(sum(...a)); // 10
```

#### Destructuring in functies

```javascript
function printRij1([rij1, ...rijen]) {
    console.log(rij1);
}

let matrix = [[1,2], [3,4], [5,6]];
printRij1(matrix); // [1,2]
```

## Objects

### Object Basics

Objects zijn verzamelingen van key-value pairs (properties).

```javascript
// Object literal
let person = { name: "tim", age: 22 };

// Properties benaderen
console.log(person.name);     // "tim" (dot notation)
console.log(person["name"]);  // "tim" (bracket notation)

// Properties wijzigen/toevoegen
person.age = 23;
person.address = "unknown";

console.log(person);
// { name: 'tim', age: 23, address: 'unknown' }
```

### Object Destructuring

Waarden uit een object uitpakken in variabelen.

```javascript
let person = { name: "tim", age: 12 };

// Enkele property uitpakken
let {age} = person;
console.log(age); // 12

// Meerdere properties uitpakken
let {name, age} = person;
console.log(name); // "tim"
console.log(age);  // 12
```

### Object Spread & Rest

#### Rest Operator - bij destructuring

```javascript
const person = {
    firstName: 'Tim',
    lastName: 'Schuermans', 
    age: 30,
    country: 'BE'
};

const {firstName, lastName, ...details} = person;

console.log(firstName); // "Tim"
console.log(lastName);  // "Schuermans"
console.log(details);   // {age: 30, country: 'BE'}
```

#### Spread Operator - bij object creatie

```javascript
const person = {
    firstName: 'Sofie',
    lastName: 'Verlaek'
};

const address = {
    street: 'Bossestraat',
    number: '57'
};

const contact = {
    ...person,
    ...address
};

console.log(contact);
// {
//   firstName: 'Sofie',
//   lastName: 'Verlaek', 
//   street: 'Bossestraat',
//   number: '57'
// }
```

## Mutable vs Immutable

### Reference Types (Mutable)

Arrays en objects zijn reference types - de variable bevat een referentie naar de data in het geheugen.

```javascript
let a = [1, 2, 3];
a[0] = 5; // Wijzigt de originele array
console.log(a); // [5, 2, 3]

let obj = {name: "Jan"};
obj.name = "Piet"; // Wijzigt het originele object
console.log(obj); // {name: "Piet"}
```

### Primitive Types (Immutable)

Strings, numbers, booleans zijn immutable - bij wijziging wordt een nieuwe waarde aangemaakt.

```javascript
let s = "abc";
s = s + "d"; // Maakt nieuwe string "abcd", originele "abc" blijft bestaan
console.log(s); // "abcd"
```

## String Methods

Strings zijn primitive types maar hebben wel methods (via automatische conversie naar String object).

```javascript
let s = "abc";

// Case conversie
s = s.toUpperCase(); // "ABC" 
s = s.toLowerCase(); // "abc"

// Zoeken
let index = s.indexOf("b"); // 1
let lastIndex = s.lastIndexOf("b"); // 1

// Substring maken
console.log("abcdef".slice(2, 4)); // "cd"
console.log("abcdef".slice(2));    // "cdef"

// Trimmen (spaties wegknippen)
console.log("  abc  ".trim()); // "abc"

// String splitsen naar array
let words = "dit is een zin".split(" ");
console.log(words); // ['dit', 'is', 'een', 'zin']

// Array samenvoegen tot string  
let numbers = [1, 2, 3];
console.log(numbers.join(",")); // "1,2,3"

// String herhalen
console.log("ha".repeat(5)); // "hahahahaha"
```

## Math Object

Het Math object bevat wiskundige constanten en functies.

```javascript
// Constanten
console.log(Math.PI); // 3.141592653589793
console.log(Math.E);  // 2.718281828459045

// Min/Max
console.log(Math.max(1, 2, 3)); // 3
console.log(Math.max(...[1, 2, 3])); // 3 (met spread)
console.log(Math.min(1, 2, 3)); // 1

// Goniometrische functies
console.log(Math.cos(Math.PI)); // -1
console.log(Math.sin(Math.PI/2)); // 1

// Random getallen (0 tot 1, exclusief 1)
function randomPointOnCircle(radius) {
    let angle = Math.random() * 2 * Math.PI;
    return {
        x: radius * Math.cos(angle),
        y: radius * Math.sin(angle)
    };
}

console.log(randomPointOnCircle(2)); // {x: 0.3667, y: 1.966}
```

## JSON (JavaScript Object Notation)

JSON is een lightweight data-interchange format voor het uitwisselen van data.

### JSON vs JavaScript Object

```javascript
// JavaScript Object
let jsObject = {
    name: "tim",
    hobbies: ["reading", "running", "tennis"]
};

// JSON String (alle keys en strings in dubbele quotes)
let jsonString = `{
    "name": "tim", 
    "hobbies": ["reading", "running", "tennis"]
}`;
```

### JSON Methods

```javascript
let person = {
    name: "tim",
    hobbies: ["reading", "running", "tennis"]
};

// Object naar JSON string
let personJSON = JSON.stringify(person);
console.log(personJSON);
// {"name":"tim","hobbies":["reading","running","tennis"]}

// JSON string naar object
let person2 = JSON.parse(personJSON);
console.log(person2);
// { name: 'tim', hobbies: [ 'reading', 'running', 'tennis' ] }
```

## Samenvatting

### Reference Types

- **Objects** en **Arrays** zijn reference types
- Ze zijn **mutable** (kunnen gewijzigd worden)
- **Object** → named properties (`obj.name`, `obj["name"]`)
- **Array** → indexed properties (`arr[0]`) + named properties (`arr.length`)

### Belangrijke Technieken

- **Destructuring**: waarden uitpakken uit arrays/objects
- **Spread operator (`...`)**: arrays/objects uitpakken
- **Rest operator (`...`)**: resterende waarden opvangen
- **JSON**: data uitwisseling tussen systemen

### Veel Gebruikte Methods

- **Array**: `push()`, `pop()`, `shift()`, `unshift()`, `slice()`, `concat()`, `indexOf()`
- **String**: `toUpperCase()`, `toLowerCase()`, `slice()`, `split()`, `join()`, `trim()`
- **Math**: `Math.random()`, `Math.max()`, `Math.min()`, `Math.PI`
- **JSON**: `JSON.stringify()`, `JSON.parse()`
