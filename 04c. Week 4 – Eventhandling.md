# Web Scripting – Eventhandling

## W3C Eventhandling

De correcte manier om events af te handelen is via de W3C standaard met `addEventListener`. **Nooit** inline eventhandling gebruiken — dat mengt HTML en JavaScript door elkaar.

```html
<!-- ❌ Niet doen: inline eventhandling -->
<a href="somewhere.html" onClick="alert('I\'ve been clicked!')">

<!-- ✅ Correct: via addEventListener in een apart JS-bestand -->
```

```js
element.addEventListener("eventtype", handlerFunction);
element.removeEventListener("eventtype", handlerFunction);
```

---

## addEventListener: basis

### Op het volledige window

```html
<p>Click this document to activate the handler.</p>
<script>
    window.addEventListener("click", () => {
        console.log("You knocked?");
    });
</script>
```

### Op een specifiek element

```html
<button>Click me</button>
<p>No handler here.</p>
<script>
    let button = document.querySelector("button");
    button.addEventListener("click", () => {
        console.log("Button clicked.");
    });
</script>
```

> De handler op `<button>` reageert alleen op klikken op de knop zelf. De `<p>` heeft geen handler.

## Het Event-object

De eventhandler ontvangt een **event-object** met informatie over het event.

```js
button.addEventListener("mousedown", event => {
    if (event.button == 0) {
        console.log("Left button");
    } else if (event.button == 1) {
        console.log("Middle button");
    } else if (event.button == 2) {
        console.log("Right button");
    }
});
```

| Eigenschap | Betekenis |
|---|---|
| `event.button` | welke muisknop (0=links, 1=midden, 2=rechts) |
| `event.buttons` | zijn er knoppen ingedrukt? (0 = geen) |
| `event.clientX` / `event.clientY` | muispositie t.o.v. scherm |
| `event.pageX` / `event.pageY` | muispositie t.o.v. document |
| `event.key` | welke toets werd ingedrukt |
| `event.target` | het element dat het event triggerde |

## Event-propagation (bubbling)

Wanneer je klikt op een element dat genest is in een ander element, reist het event **omhoog** door de DOM-tree (bubbling). Handlers van dieper geneste elementen worden eerst uitgevoerd.

```
<button> → <p> → <body> → <html>
```

**`event.stopPropagation()`** stopt de opwaartse verspreiding.

```js
let para = document.querySelector("p");
let button = document.querySelector("button");

para.addEventListener("mousedown", () => {
    console.log("Handler for paragraph.");
});

button.addEventListener("mousedown", event => {
    console.log("Handler for button.");
    if (event.button == 2) {
        event.stopPropagation(); // rechtsklik stopt bij button
    }
});
```

> Bij een linksklik op `<button>` verschijnt eerst "Handler for button.", daarna "Handler for paragraph." Bij een rechtsklik stopt het bij de button.

## Target

`event.target` geeft het element dat het event triggerde. Handig bij één listener op een parent voor meerdere children.

```html
<button>A</button>
<button>B</button>
<button>C</button>
<script>
    document.body.addEventListener("click", event => {
        if (event.target.nodeName == "BUTTON") {
            console.log("Clicked", event.target.firstChild.nodeValue);
        }
    });
</script>
// → Clicked A / Clicked B / Clicked C
```

> Eén listener op `document.body` afhandelt klikken op alle knoppen via bubbling.

## Default actions

Veel HTML-elementen hebben een standaardgedrag bij een event (bijv. `<a>` navigeert bij een klik). **`event.preventDefault()`** onderbreekt dat gedrag.

```js
let link = document.querySelector("a");
link.addEventListener("click", event => {
    console.log("Nope.");
    event.preventDefault(); // link navigeert niet
});
```

## Key events

```js
// keydown: toets ingedrukt / keyup: toets losgelaten
window.addEventListener("keydown", event => {
    if (event.key == "v") {
        document.body.style.background = "violet";
    }
});

window.addEventListener("keyup", event => {
    if (event.key == "v") {
        document.body.style.background = "";
    }
});
```

### Modifier keys

```js
window.addEventListener("keydown", event => {
    if (event.key == " " && event.ctrlKey) {
        console.log("Continuing!"); // Ctrl+Spatie
    }
});
```

| Eigenschap | Betekenis |
|---|---|
| `event.key` | de ingedrukte toets als string (bijv. `"v"`, `"Enter"`, `" "`) |
| `event.shiftKey` | is Shift ingedrukt? |
| `event.ctrlKey` | is Ctrl ingedrukt? |
| `event.altKey` | is Alt ingedrukt? |

## Mouse clicks

`event.pageX` en `event.pageY` geven de muispositie t.o.v. het document.

```html
<style>
    body { height: 200px; background: beige; }
    .dot {
        height: 8px; width: 8px;
        border-radius: 4px;
        background: blue;
        position: absolute;
    }
</style>
<script>
    window.addEventListener("click", event => {
        let dot = document.createElement("div");
        dot.className = "dot";
        dot.style.left = (event.pageX - 4) + "px";
        dot.style.top  = (event.pageY - 4) + "px";
        document.body.appendChild(dot);
    });
</script>
```

> Bij elke klik wordt een blauw bolletje op de klikpositie toegevoegd.

## Mouse move: drag-to-resize

```html
<div style="background: orange; width: 60px; height: 20px"></div>
<script>
    let lastX;
    let bar = document.querySelector("div");

    bar.addEventListener("mousedown", event => {
        if (event.button == 0) {
            lastX = event.clientX;
            window.addEventListener("mousemove", moved);
            event.preventDefault(); // voorkom tekst-selectie
        }
    });

    function moved(event) {
        if (event.buttons == 0) {
            // geen knop ingedrukt: stop listener
            window.removeEventListener("mousemove", moved);
        } else {
            let dist = event.clientX - lastX;
            let newWidth = Math.max(10, bar.offsetWidth + dist);
            bar.style.width = newWidth + "px";
            lastX = event.clientX;
        }
    }
</script>
```

> Bij `mousedown` op de oranje balk start het bijhouden van de muisbeweging. Bij `mousemove` wordt de breedte aangepast. Wanneer geen knop meer ingedrukt is (`event.buttons == 0`), wordt de listener verwijderd.

## Focus event

`focus` treedt op wanneer een element focus krijgt (bijv. klikken in een invoerveld). `blur` treedt op wanneer het element focus verliest.

```html
<p>Name: <input type="text" data-help="Your full name"></p>
<p>Age: <input type="text" data-help="Your age in years"></p>
<p id="help"></p>
<script>
    let help = document.querySelector("#help");
    let fields = document.querySelectorAll("input");
    let textNodeData;

    for (let field of Array.from(fields)) {
        field.addEventListener("focus", event => {
            let text = event.target.getAttribute("data-help");
            textNodeData = document.createTextNode(text);
            help.appendChild(textNodeData);
        });
        field.addEventListener("blur", event => {
            help.replaceChild(document.createTextNode(""), textNodeData);
        });
    }
</script>
```

> Bij focus op een veld verschijnt de helptekst uit `data-help`. Bij blur verdwijnt die weer.

## Load event

Tot nu toe stond `<script>` onderaan in het HTML-document zodat alle elementen al geladen waren. De betere aanpak is wachten op het **`load`-event** van `window` en dan pas eventlisteners toevoegen.

```html
<button>click</button>
<script>
    const handleLoad = () => {
        let button = document.querySelector("button");
        button.addEventListener("click", handleClick);
    };

    const handleClick = (event) => {
        console.log(event.target.firstChild.nodeValue);
    };

    window.addEventListener("load", handleLoad);
</script>
```

> `handleLoad` wordt pas uitgevoerd als de volledige pagina geladen is. Dan pas wordt de click-listener op de button gezet.

## Samenvatting

| Concept | Hoe |
|---|---|
| Event toevoegen | `element.addEventListener("type", handler)` |
| Event verwijderen | `element.removeEventListener("type", handler)` |
| Muisknop | `event.button` (0/1/2) |
| Muispositie scherm | `event.clientX` / `event.clientY` |
| Muispositie document | `event.pageX` / `event.pageY` |
| Toets | `event.key` |
| Modifier keys | `event.shiftKey`, `event.ctrlKey`, `event.altKey` |
| Welk element triggerde | `event.target` |
| Bubbling stoppen | `event.stopPropagation()` |
| Standaardgedrag stoppen | `event.preventDefault()` |
| Pagina geladen | `window.addEventListener("load", ...)` |
| Veelgebruikte events | `click`, `mousedown`, `mousemove`, `keydown`, `keyup`, `focus`, `blur`, `load` |
