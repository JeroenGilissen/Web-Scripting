# Web Scripting – The Document Object Model (DOM)

## Wat is de DOM?

Wanneer een pagina geopend wordt in de browser:
1. De **parser** ontleedt de HTML-code
2. Een **boomstructuur** van de inhoud wordt gebouwd
3. Aan de hand van die boomstructuur wordt een grafische weergave gegenereerd
4. De boomstructuur kan **opgevraagd en gewijzigd** worden via Javascript

---

## DOM-tree: boomstructuur van nodes

```html
<!doctype html>
<html>
    <head>
        <title>My home page</title>
    </head>
    <body>
        <h1>My home page</h1>
        <p>Hello, I am Marijn and this is my home page.</p>
        <p>I also wrote a book! Read it
            <a href="http://eloquentjavascript.net">here</a>.
        </p>
    </body>
</html>
```

De DOM-tree bestaat uit twee soorten nodes:
- **ELEMENT_NODE** – een HTML-element zoals `<p>`, `<h1>`, `<a>`
- **TEXT_NODE** – de tekst binnenin een element

### Relaties tussen nodes

- `<html>` is de **parent** van `<head>` en `<body>`
- `<head>` en `<body>` zijn **children** van `<html>`
- `<head>` en `<body>` zijn **siblings** (generatiegenoten)

---

## Navigatie in de DOM-tree

Via eigenschappen van een node navigeer je door de boom:

| Eigenschap | Betekenis |
|---|---|
| `.childNodes` | alle kinderen van een node |
| `.firstChild` | eerste kind |
| `.lastChild` | laatste kind |
| `.parentNode` | de ouder |
| `.previousSibling` | vorige broer/zus |
| `.nextSibling` | volgende broer/zus |

**Voorbeeld:** `<body>` heeft `childNodes`: `<h1>`, `<p>`, `<p>`, `firstChild`: `<h1>`, `lastChild`: `<p>`

## Recursief de DOM doorzoeken

```js
// zoek.js – zoek het woord "book" in document.body
function talksAbout(node, searchString) {
    if (node.nodeType == document.ELEMENT_NODE) {
        for (let i = 0; i < node.childNodes.length; i++) {
            if (talksAbout(node.childNodes[i], searchString)) {
                return true;
            }
        }
        return false;
    } else if (node.nodeType == document.TEXT_NODE) {
        return node.nodeValue.indexOf(searchString) != -1;
    }
}

console.log(talksAbout(document.body, "book"));
```

> Als de node een **ELEMENT_NODE** is, worden alle kindelementen recursief doorlopen.
> Als de node een **TEXT_NODE** is, wordt gekeken of de tekst de zoekstring bevat. `indexOf()` geeft -1 terug als de string niet gevonden wordt.

## Live lists vs Static lists

Bij DOM-zoekopdrachten krijg je een **live** of **static** lijst terug. Beide zijn iterable (bruikbaar in een for-loop).

**Live list (HTMLCollection)** – update zichzelf automatisch bij wijzigingen in de DOM-tree.

```js
let list = document.getElementsByTagName('p'); // live HTMLCollection
// childNodes ophalen geeft ook een live lijst
```

**Static list (NodeList)** – verandert niet na aanpassingen aan de DOM-tree.

```js
let list = document.querySelectorAll('p'); // static NodeList
// childNodes geeft ook een static NodeList
```

**Live list omzetten naar static (array):**

```js
let list = Array.from(document.getElementsByTagName('p')); // nu static
```

## DOM: elementen zoeken

### `getElementById`
Zoek één element op basis van het `id`-attribuut.

```js
let ostrich = document.getElementById("gertrude");
console.log(ostrich.src); // img/ostrich.png
```

### `getElementsByTagName`
Geeft een **live HTMLCollection** terug van alle elementen met een bepaalde tag.

```js
let firstLink = document.getElementsByTagName("a")[0];
console.log(firstLink.href); // http://destandaard.be
// [0] = selecteer de eerste uit de rij
```

### `getElementsByClassName`
Geeft een **live HTMLCollection** terug van elementen met een bepaalde class.

```js
let firstRed = document.getElementsByClassName("red")[0];
console.log(firstRed.textContent); // My home page
```

### `querySelector`
Selecteer **1 element** aan de hand van een CSS-selector (het eerste dat voldoet).

```js
let element = document.querySelector("p");          // eerste <p>
let element2 = document.querySelector(".red");       // eerste met class="red"
let element3 = document.querySelector("#mijnId");    // element met id="mijnId"
```

> ⚠️ Een id gebruik je met `#`, een class met `.` — net zoals in CSS.

### `querySelectorAll`
Selecteer **alle elementen** die voldoen aan een CSS-selector → **static NodeList**.

```js
let elements = document.querySelectorAll("p");       // alle <p>
let elements2 = document.querySelectorAll(".red");   // alle met class="red"
```

**Voorbeeld met CSS-selectors:**

```js
function count(selector) {
    return document.querySelectorAll(selector).length;
}

console.log(count("p"));           // → 4   (alle <p>)
console.log(count(".animal"));     // → 2   (class="animal")
console.log(count("p .animal"));   // → 2   (class="animal" onder een <p>)
console.log(count("p > .animal")); // → 1   (directe child van <p>)
```

## DOM: aanpassingen

```js
parentElement.appendChild(newElement);              // voeg toe achteraan
parentElement.insertBefore(newElement, existingElement); // voeg in voor element
parentElement.replaceChild(newElement, existingElement); // vervang element
```

> ⚠️ Elke node kan maar **één keer** voorkomen in de boomstructuur. Bij `insertBefore` met een bestaande node verdwijnt die automatisch op de oude plek.

**Voorbeeld – `insertBefore`:**

```html
<p>One</p>
<p>Two</p>
<p>Three</p>
<script>
    let paragraphs = document.body.getElementsByTagName("p");
    document.body.insertBefore(paragraphs[2], paragraphs[0]);
    // Resultaat: Three, One, Two
</script>
```

## Nodes aanmaken

### `createTextNode` – maak een tekstnode aan

```js
let node = document.createTextNode(text);
```

**Voorbeeld – vervang afbeeldingen door hun alt-tekst:**

```js
function replaceImages() {
    let images = document.getElementsByTagName("img");
    for (let i = images.length - 1; i >= 0; i--) {
        let image = images[i];
        if (image.alt) {
            let text = document.createTextNode(image.alt);
            image.parentNode.replaceChild(text, image);
        }
    }
}
```

> Loop achteraan beginnen (`i--`) omdat `getElementsByTagName` een **live list** is die verkort bij elke `replaceChild`.

### `createElement` – maak een nieuw element aan

```js
let element = document.createElement(name);
```

**Voorbeeld – hulpfunctie `elt` die een element met kinderen aanmaakt:**

```js
function elt(name, ...children) {
    let element = document.createElement(name);
    for (let child of children) {
        if (typeof child == "string") {
            element.appendChild(document.createTextNode(child));
        } else {
            element.appendChild(child);
        }
    }
    return element;
}

// Gebruik:
document.getElementById("quote").appendChild(
    elt("footer",
        "-",
        elt("strong", "Karl Popper"),
        ", preface to the second edition of ",
        elt("em", "The Open Society and Its Enemies"),
        ", 1950"
    )
);
```

## Attributen

Sommige attributen zijn rechtstreeks beschikbaar als eigenschap (bijv. `element.href`). Voor alle andere gebruik je:

```js
element.getAttribute("data-classified"); // lees attribuut
element.setAttribute("data-classified", "secret"); // schrijf attribuut
```

**Voorbeeld – verwijder geheime paragrafen:**

```js
let paras = document.body.getElementsByTagName("p");
for (let para of paras) {
    if (para.getAttribute("data-classified") == "secret") {
        para.remove();
    }
}
```

## Style: width & height

Voor **block elements** kan je de afmetingen opvragen via:

| Eigenschap | Wat |
|---|---|
| `clientWidth` / `clientHeight` | zonder padding & border |
| `offsetWidth` / `offsetHeight` | met padding & border, zonder margin |
| `getBoundingClientRect()` | geeft een DOMRect terug (width, height, top, left, ...) relatief t.o.v. het scherm |
| `pageXOffset` / `pageYOffset` | positie relatief t.o.v. het document |

```js
let para = document.body.getElementsByTagName("p")[0];
console.log("clientHeight:", para.clientHeight); // zonder border
console.log("offsetHeight:", para.offsetHeight); // met border

let rect = para.getBoundingClientRect();
console.log(rect.height);
```

## Style: CSS via JS aanpassen

Via `.style` kun je de inline CSS van een element lezen en schrijven.

```html
<p id="para" style="color: purple">Nice text</p>
<script>
    let para = document.getElementById("para");
    console.log(para.style.color); // purple
    para.style.color = "magenta";  // tekst wordt magenta
</script>
```

## Positioning & Animating

### CSS `position`-waarden

| Waarde | Betekenis |
|---|---|
| `static` | normale positie, blijft in document flow, top/left/... worden genegeerd |
| `relative` | gepositioneerd t.o.v. de normale (static) positie, blijft in flow |
| `absolute` | gepositioneerd t.o.v. dichtstbijzijnde niet-static ancestor, verdwijnt uit flow |

### Animatie met `requestAnimationFrame`

```js
let cat = document.querySelector("img");
let angle = Math.PI / 2;

function animate(time, previousTime) {
    if (previousTime != null) {
        angle += (time - previousTime) * 0.001;
    }
    cat.style.top  = (Math.sin(angle) * 20)  + "px";
    cat.style.left = (Math.cos(angle) * 200) + "px";
    requestAnimationFrame(newTime => animate(newTime, time));
}

requestAnimationFrame(animate);
```

> `requestAnimationFrame` roept de meegegeven functie aan vóór de volgende schermupdate, met een **timestamp** als argument. Zo wordt de animatiesnelheid gekoppeld aan de verstreken tijd, niet aan de framerate.
