## Symbol (extra)

Een **Symbol** is een primitief datatype waarbij elke aanroep van `Symbol()` een **unieke waarde** produceert — ook al geef je dezelfde beschrijving mee.

```js
let symbol = Symbol("something");
console.log(symbol == Symbol("something")); // false
```

Symbols worden gebruikt als keys in objecten of klassen. Het grote nut: **name collisions vermijden**. Twee verschillende code-stukken kunnen nooit per ongeluk dezelfde Symbol-key gebruiken.

```js
let id = Symbol("id");

let person = {
  name: 'tim',
  [id]: 1        // symbool als key → gebruik blokhaken [ ]
};

console.log(person.name);  // tim
console.log(person[id]);   // 1
```

Ook als functienaam:

```js
let print = Symbol("print");

let person = {
  name: 'tim',
  [print]: function () {
    console.log(this.name);
  }
};

person[print](); // tim
```

## Symbol: iterator (extra)

`Symbol.iterator` is een ingebouwd, globaal uniek Symbol dat JavaScript gebruikt om een object **itereerbaar** te maken. Als een object deze methode heeft, kan je er `for...of` op gebruiken.

Strings bevatten reeds een `Symbol.iterator`, waardoor je karakter per karakter kunt itereren:

```js
let name = "tim";

for (let value of name) {
  console.log(value);
}
// t
// i
// m
```

## Voorbeeld: Matrix met iterator (extra)

Een matrix (2D tabel) wordt intern bewaard als **1D array**. De omzetting werkt als volgt:

```js
height = 3, width = 2

       j=0  j=1
i=0  [  a ,  b  ]   →  index 0, 1
i=1  [  c ,  d  ]   →  index 2, 3
i=2  [  e ,  f  ]   →  index 4, 5

index = i * width + j
i     = Math.floor(index / width)
j     = index % width
```

### De Matrix klasse

```js
class Matrix {
  constructor(height, width, element = (i, j) => undefined) {
    this.height = height;
    this.width = width;
    this.contents = [];

    for (let i = 0; i < height; i++) {
      for (let j = 0; j < width; j++) {
        this.contents[i * this.width + j] = element(i, j);
      }
    }
  }

  get(i, j) {
    return this.contents[i * this.width + j];
  }

  set(i, j, value) {
    this.contents[i * this.width + j] = value;
  }

  // Maakt Matrix itereerbaar via for...of
  [Symbol.iterator]() {
    return new MatrixIterator(this);
  }
}
```

### De MatrixIterator klasse

Een iterator moet een `next()` methode hebben die telkens het volgende element teruggeeft als `{ value, done: false }`, of `{ done: true }` als alles doorlopen is.

```js
class MatrixIterator {
  constructor(matrix) {
    this.index = 0;
    this.matrix = matrix;
  }

  next() {
    if (this.index == this.matrix.contents.length) {
      return { done: true };
    }

    let i = Math.floor(this.index / this.matrix.width);
    let j = this.index % this.matrix.width;
    let value = { i, j, element: this.matrix.get(i, j) };
    this.index++;

    return { value, done: false };
  }
}
```

### Gebruik

```js
let matrix = new Matrix(3, 2, (i, j) => i * 2 + j);

for (let { i, j, element } of matrix) {
  console.log(i, j, element);
}
// 0 0 0
// 0 1 1
// 1 0 2
// 1 1 3
// 2 0 4
// 2 1 5
```

> 💡 Door `[Symbol.iterator]()` te definiëren in de klasse, begrijpt JavaScript dat `for...of` de `MatrixIterator` moet gebruiken. Dit is het patroon achter alle ingebouwde itereerbare types (Array, String, Map, ...).

## Overzicht

| Concept | Sleutelwoord / syntax |
|---|---|
| Unieke key aanmaken | `Symbol("beschrijving")` |
| Symbol als key in object | `{ [mySymbol]: value }` |
| Object itereerbaar maken | `[Symbol.iterator]() { return iterator; }` |
| Iterator interface | `next()` → `{ value, done }` |
| Itereren over object | `for...of` |
